HackBox is an all in one tool for lazy hackers. It is written in python and it runs in both rooted and non-rooted devices.
***
## [+] Installation and Usage
* $ apt update
* $ apt upgrade
* $ pkg update
* $ pkg upgrade
* $ pkg install git
* $ pkg install python
* $ pkg install python2
* $ git clone https://github.com/HackWeiser360/HackBox
* $ cd HackBox
* $ python HackBox.py
